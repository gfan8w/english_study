# **solicit**

英  [səˈlɪsɪt]
****  美  [səˈlɪsɪt]

- v. 请求，索求，征求（意见）；<美>兜售，推销；拉（客），卖淫

an unsolicited payment return can be generated only for credit transfer

一个未经同意的支付反还.....

"An unauthorized payment" 指的是未经授权或未经同意的支付，即在没有获得允许或许可的情况下进行的付款。这可能是指在没有获得持卡人或账户所有者的明确同意的情况下进行的转账或支付。

而 "an unsolicited payment" 指的是未经请求或索求的支付。它强调的是没有被要求或需要的情况下，某人或某实体向他人进行了付款。这可能是由于误操作、系统错误、慷慨捐赠或其他无需被索求的原因。

因此，虽然两者都表示未经同意的支付，但 "an unauthorized payment" 更强调没有合法授权或许可，而 "an unsolicited payment" 则侧重于没有被请求或需要。
